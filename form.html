2. TERMO 1 & FÍSICO-QUÍMICA BASE

Regra das Fases de Gibbs: F = 2 - π + N
Definições Fundamentais: H = U + PV
A = U - TS
G = H - TS
Relações de Propriedades (Sist. Fechado): dU = TdS - PdV
dH = TdS + VdP
dG = -SdT + VdP
Capacidade Térmica: dH = Cp dT   |   dU = Cv dT

3. CAP 1: EQUILÍBRIO DE FASES (PURA)

Critério de Equilíbrio de Fases: Tα = Tβ   |   Pα = Pβ   |   μα = μβ  (ou  gα = gβ)
Transição de Fase (Critério G e S): Δgtr = 0  ⇒  Δhtr = TΔstr
Δstr = Δhtr / T
Equação de Clapeyron (Geral): dPsat / dT = Δhtr / (T · Δvtr)
Eq. Clausius-Clapeyron (L-V ou S-V): *Hipóteses: Vapor como GI (vv = RT/P), vl desprezível, Δhvap constante. ln(P2sat / P1sat) = -(Δhvap / R) · [ (1 / T2) - (1 / T1) ]

d(ln Psat) / d(1/T) = - Δhvap / R
Equação de Antoine: ln Psat = A - [ B / (T + C) ]

4. CAP 2: TERMODINÂMICA DE MISTURAS

Definição Propriedade Parcial Molar: Mi = [ ∂(nM) / ∂ni ]T, P, nj≠i
Relação de Soma (Mistura): M = Σ xi Mi
Equação de Gibbs-Duhem: Σ xi dMi = 0   (a T, P constantes)
Cálculo para Misturas Binárias (1 e 2): M1 = M + x2 · (dM / dx1)
M2 = M - x1 · (dM / dx1)
M1 = M - x2 · (dM / dx2)
Potencial Químico: μi = Gi
Propriedade de Mistura (ΔMmis): ΔMmis = M - Σ xi Mi
Mistura de Gás Ideal (GI): ViGI = ViGI   |   HiGI = HiGI
ΔVmisGI = 0   |   ΔHmisGI = 0

GiGI = μiGI = GiGI + RT ln(yi)
GGI = Σ yi GiGI + RT Σ yi ln(yi)
ΔGmisGI = RT Σ yi ln(yi)
ΔSmisGI = -R Σ yi ln(yi)

VERSO - MATEMÁTICA E CONVERSÕES

1. CONSTANTES UNIVERSAIS

R (Constante dos Gases)
= 8,314 J / (mol·K)
= 8,314 (m³·Pa) / (mol·K)
= 0,08314 (L·bar) / (mol·K)
= 82,06 (cm³·atm) / (mol·K)
= 1,987 cal / (mol·K)

2. CONVERSÃO DE UNIDADES

Pressão
1 atm = 101325 Pa = 1,01325 bar
1 bar = 105 Pa = 100 kPa
1 atm = 760 mmHg = 760 torr
Volume & Energia
1 L = 10-3 m³ = 1000 cm³ = 1000 mL
1 cm³ = 10-6 m³
1 J = 1 Pa·m³ = 10 cm³·bar = 10-3 kJ
Temperatura & Massa Molar
T (K) = T (°C) + 273,15
MM (kg/mol) = MM (g/mol) / 1000

3. REGRAS DE LOGARITMOS E EXPONENCIAIS

• ln(a · b) = ln(a) + ln(b)

• ln(a / b) = ln(a) - ln(b)

• ln(ab) = b · ln(a)

• ln(e) = 1   |   ln(1) = 0

• eln(x) = x   |   ln(ex) = x

4. DERIVADAS FUNDAMENTAIS

• Constante: d(c)/dx = 0

• Potência: d(xn)/dx = n · xn-1

• Constante x Função: d(c·u)/dx = c · (du/dx)

• Soma: d(u ± v)/dx = (du/dx) ± (dv/dx)

• Produto: d(u·v)/dx = u · (dv/dx) + v · (du/dx)

• Quociente: d(u/v)/dx = [ v · (du/dx) - u · (dv/dx) ] / v2

• Logarítmica: d(ln x)/dx = 1 / x

• Regra da Cadeia: d(ln u)/dx = (1 / u) · (du/dx)

5. INTEGRAIS FUNDAMENTAIS

• ∫ dx = x + C

• ∫ xn dx = [ xn+1 / (n + 1) ] + C   (para n ≠ -1)

• ∫ (1 / x) dx = ln|x| + C

• Integração Definida: ∫AB f(x) dx = F(B) - F(A)

• Capacidade Térmica (Exemplo de aplicação):
   ∫ (A + BT + CT2) dT = AT + (B/2)T2 + (C/3)T3

6. MANIPULAÇÕES ALGÉBRICAS ÚTEIS

• Soma de Frações Molares:
   Σ xi = 1  ⇒  x1 + x2 = 1  ⇒  dx2 = -dx1

• Propriedades Parciais a partir de Frações:
   Se M = f(x1), então dM/dx1 é a derivada de M em relação a x1.
   Lembre-se de substituir x2 por (1 - x1) se necessário antes de derivar.
